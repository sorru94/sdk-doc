Features (multiple-file)ÔÉÅ

A set of possible features supported by the SDKs is presented in this section. Each Astarte device SDK implements a slighly different set of features. While some might be common to all the SDKs, others might be more relevant to some programming language or framework and subequently be only implemented in a set of SDKs.

Here is a comprehensive list of features for the Astarte device SDKs:

  • Device ID generation
  • Device registration
    • On board device registration
  • Interfaces parsing
  • Device connection
    • Server TLS certificates support
    • Client TLS certificate support
    • Client TLS certificate renewal
    • Reconnection
  • Data streaming
    • Individual data transmission and reception
    • Aggregated data transmission and reception
    • Reliability support
    • Data persistence and automatic retransmission
  • Properties
    • Setting/unsetting device properties
    • Setting/unsetting server properties
    • Properties presistency
  • Data Validation
